# vunostudio

Site institucional Vuno Studio com painel administrativo.

## Stack

- HTML5
- CSS3
- JavaScript puro
- Supabase (Auth + Database)

## Estrutura

- `index.html` - Página inicial
- `servicos.html` - Página de serviços
- `planos.html` - Página de planos
- `admin.html` - Painel administrativo
- `data/` - Dados JSON (fallback)
- `js/` - Scripts do site e admin
- `css/` - Estilos
- `images/` - Assets

## Deploy

Este projeto está configurado para deploy em plataformas de hospedagem estática (Vercel, Netlify, Cloudflare Pages).

## Banco de Dados

O conteúdo dinâmico é gerenciado via Supabase na tabela `site_data`.
